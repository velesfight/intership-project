- React, Typescript, Vite, Radix UI, CSS Modules
- см. другие зависимости и скрипты в [package.json](package.json)

## Требования

- NodeJS 20+, NPM
- [рекомендуемые расширения для VSCode](.vscode/extensions.json)

## Запуск

1. `git clone https://gitlab.ghfls.ru/root/nude_web_v2`
2. в директории проекта создать env файл с переменной, которая содержит api url: `echo "VITE_API_URL_BASE = 'https://dev-back-nude-sdhf43sd5f-v2.ghfls.ru'" > .env`
3. `npm i`
4. `npm run dev`
